كيفية استخدام API

لاستخدام API، قم بإضافة الترويسات التالية إلى طلبك:

X-API-Key: your-api-key-here
X-API-Name: your-api-name-here

curl -X POST \
  http://localhost:5000/ocr \
  -H 'X-API-Key: {{ static_key.key }}' \
  -H 'X-API-Name: {{ static_key.name }}' \
  -F 'file=@/path/to/your/file.pdf'
